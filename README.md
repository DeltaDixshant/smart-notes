# Smart Notes

A Flask-based note-taking application with CI/CD pipeline.

## Features
- User authentication
- Create, read, update, delete notes
- Responsive design
- Security scanning with OWASP ZAP
- Code quality analysis with SonarCloud

## Tech Stack
- Python 3.9
- Flask
- SQLAlchemy
- Docker

## CI/CD Pipeline
- Automated testing
- SonarCloud code analysis
- Security scanning
- Docker containerization
